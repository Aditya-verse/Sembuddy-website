# ✨ About SemBuddy

**SemBuddy** is a powerful, all-in-one platform designed exclusively for **engineering students** to simplify and supercharge their academic life.

No more manual calculations, lost timetables, or digging through PDFs — SemBuddy puts everything in one clean, modern interface.

> _"The smartest engineering students don't just work hard — they use SemBuddy."_

Whether you need to calculate your internal marks, track attendance, browse your syllabus, or get instant AI-powered academic help — SemBuddy has you covered.

---

# 🔥 Key Features

### 📊 Marks Calculator
Instantly calculate your internal, external, and semester marks with precision. No spreadsheets needed.

### 📅 Attendance Tracker
Track your attendance percentage subject-wise. Get alerts when you're close to the minimum limit.

### 📚 Syllabus Viewer
Access the complete, up-to-date syllabus for all engineering subjects — organized and easy to browse.

### 🤖 AI Study Assistant
Ask academic questions and get instant, intelligent answers powered by **Gemini AI** — available 24/7.

### 📱 Android App
Download the native SemBuddy Android app for an even smoother on-the-go experience.

# 💻 Run Locally

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Aditya-verse/Sembuddy-website.git
```

### 2️⃣ Navigate to the project
```bash
cd Sembuddy-website
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Add your API key
Create a `.env.local` file in the root:
```env
GEMINI_API_KEY=your_api_key_here
```

### 5️⃣ Start the development server
```bash
npm run dev
```

Open in browser:
```
http://localhost:5173
```
